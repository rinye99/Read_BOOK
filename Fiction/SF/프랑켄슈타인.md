## 🌟 기억에 남는 구절

*p.22* <br/>
*누님, 제게는 친구가 없습니다.* <br/>
*성공을 향한 열정으로 가슴이 벅차올라도 함께 기뻐해줄 사람이 없고 절망에 빠져도 위로해줄 이가 없다는 뜻이죠.*

*p.37* <br/>
*낯 뜨거운 표현에 얼마든지 웃으셔도 좋아요.*<br/>
*저는 그래도 매일 그를 치켜세울 새로운 이유를 찾을 거랍니다.*

*p.67* <br/>
*새로 탄생한 종족은 나를 창조자이자 근원으로 축복할 것이고 나로 말미암아 행복하고 탁월한 존재들이 수없이 생겨나리라고 상상했어요.*<br/>
*나는 자식을 낳은 아버지와는 비교도 할 수 없을 만큼 완벽한 보람을 느낄 테고요.*

*p.97* <br/>
*그대의 봉우리는 선명하고 하늘과 호수는 잔잔하고 푸르구나.*<br/>
*평화의 조짐인가 아니면 나의 불행을 조롱하는건가?*

*p.101* <br/>
*마치 나 자신의 흡혈귀 같았지요.*<br/>
*나의 혼령이 무덤에서 나와 내게 소중한 이들을 모두 파멸하려 드는 것 같았습니다.*

*p.116* <br/>
*"저는 이 부당하고 더러운 세상의 근심에서 벗어나 하늘에서 더 행복한 삶을 살 거예요.*<br/>
*그런 생각으로 저를 보내주세요. 나의 훌륭한 친구, 저를 절망에 빠뜨리지 말아주세요."*

*p.123* <br/>
*잇따른 사건을 겪으며 수많은 감정이 소용돌이치고 나면 죽음처럼 고요한 무위와 확신이 희망뿐 아니라 두려움까지 앗아 가버린답니다.*<br/>
*그보다 더 인간의 마음을 괴롭히는 것은 없지요.*<br/>
*쥐스틴은 죽어서 안식을 찾았지만 나는 살아 있었습니다.*<br/>
*내 혈관에는 여전히 피가 돌았지만 가슴을 짓누르는 절망과 가책을 무엇으로도 덜어낼 수 없었습니다.*

*p.124* <br/>
*나의 유일한 위안은 고독 뿐이었습니다. 깊고 어두운, 죽음 같은 고독.*

*p.133* <br/>
*아! 어째서 인간은 짐승보다 우월한 감성을 지녔을까요?*<br/>
*그래봐야 더욱 얽매이기만 할 뿐인데.*

*p.136* <br/>
*삶은 고통의 연속이지만 그래도 나에겐 소중하니 내 삶을 지킬 것이다.*

*p.165* <br/>
*그렇다면 나는 괴물일까? 지상의 오점일까?*<br/>
*그래서 모든 인간이 피하고 도망치는 것일까?*

*p.166* <br/>
*하지만 고통을 이기는 방법은 한 가지뿐이라는 것을 깨달았다. 바로 죽음이었어.*

*p.179* <br/>
*'내가 생명을 얻게 된 날은 얼마나 끔찍한가! 저주받을 창조자!*<br/>
*어째서 스스로도 혐오감에 고개를 돌릴 만큼 끔찍한 괴물을 만들었단 말인가?*<br/>
*신은 자신의 모습을 본떠 아름잡고 매혹적인 인간을 만들었는데,*<br/> 
*나는 인간을 본떴음에도 추악하고 오히려 인간과 비슷해서 더 진저리 나는 형상이 됐지.'*

*p.199* <br/>
*"하지만 나처럼 끔찍하고 흉한 여자라면 나를 밀어내지 않을 거야.*
*나와 같은 부류, 나와 같은 결함을 가진 동반자가 필요해. 그런 존재를 만들어줘."*

*p.314* <br/>
*타락한 천사는 사악한 악마가 되는법.*<br/>
*그러나 신과 인간의 적인 타락 천사에게도 외로움을 나눌 친구가 있었지만 나에겐 아무도 없다.*
