## 🌟 기억에 남는 구절

### 🔸 동물 농장 

*p.14* <br/>
*"두 발로 걷는 것은 모두 우리의 적입니다.* <br/>
*네 발로 걷거나 날개를 가진 것은 모두 우리의 친구입니다.* <br/>
*인간에 맞서 싸울 때 우리 동물들이 결코 인간을 닮아서는 안 된다는 점도 기억하시오."*

*p.28* <br/>
*[일곱 계명]* <br/>

*1. 무엇이건 두 발로 걷는 것은 적이다.* <br/>
*2. 무엇이건 네 발로 걷거나 날개를 가진 것은 친구다.* <br/>
*3. 어떤 동물도 옷을 입어서는 안 된다.* <br/>
*4. 어떤 동물도 침대에서 자서는 안 된다.* <br/>
*5. 어떤 동물도 술을 마시면 안 된다.* <br/>
*6. 어떤 동물도 다른 동물을 죽여선 안 된다.* <br/>
*7. 모든 동물은 평등하다.* <br/>

*p.45* <br/>
*스노볼 : "유일하게 좋은 인간은 죽은 인간이오"*

*p.125* <br/>
*어쩌면 지금 생존해 있는 동물들의 살아생전에는 오지 않을지 몰라도, 그래도 그날은 오고 있었다.*

*p.126* <br/>
*돼지 하나가 두 발로 서서 걷고 있었다.*

*p.127* <br/>
*그러나 바로 그때, 무슨 신호를 받기라도 한 듯 양들이 일제히 목청을 높여 우렁차게 외혀 대기 시작했다.* <br/>
*"네 발은 좋고 두 발은 더 좋다! 네 발은 좋고 두 발은 더 좋다! 네 발은 좋고 두 발은 더 좋다! 네 발은 좋고 두 발은 더 좋다!"*

*p.128* <br/>
*일곱 계명은 오간 데 없고 단 하나의 계명만이 거기 적혀 있었다.*<br/>

*모든 동물은 평등하다.*<br/>
*그러나 어떤 동물은 다른 동물들보다 더 평등하다.*<br/>

*p.135* <br/>
*그러나 누가 돼지고 누가 인간인지, 어느 것이 어느 것인지 이미 분간할 수 없었다.*

<br/>

### 🔸 나는 왜 쓰는가

*p.143* <br/>
*아주 어릴 때부터, 그러니까 다섯 살이나 여섯 살 때부터 나는 내가 커서 작가가 될 것임을 알았다.*<br/>

*p.147* <br/>
*먹고 살아야 한다는 요구를 제외한다면, 나는 작가들이 글을 쓰게 되는 데는 네 가지 큰 동기들이 있다고 생각한다.*<br/>

*1. **순전한 이기심**. 남들보다 똑똑해 보이고 사람들의 입에 오르내리며 죽은 후에도 기억되고 어린 시절 자기를 무시했던 어른들에게 보복하고 싶은 욕망.* <br/>
*2. **미학적 열정**. 어떤 경험을 공유해 보려는 욕망* <br/>
*3. **역사적 충동**. 후대를 위해 이것들이 모아 두려는 욕망.* <br/>
*4. **정치적 목적**. 세계를 특정 방향으로 밀고 가려는 욕망.* <br/>

## 🌟 어땠냐?
* 1984에 이후 조지 오웰의 소설을 두 번째로 읽었는 데, 재미있었다. 조지 오웰 작품이 잘 맞는 것 같다.
* 돼지가 주도하는 이상 완전한 유토피아는 없다.
